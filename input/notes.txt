Проект: компьютерная игра The Hero в жанре пошаговой стратегии/тактики.

Основной игровой цикл: карта -> событие/бой -> награда -> усиление армии/базы.

Ключевые функции:
- главное меню: новая игра, продолжить, настройки, выход;
- карта мира: тайловая сетка, выбор цели, расчет маршрута, расход очков движения;
- взаимодействие с объектами: ресурсы, нейтральные противники, статичные противники, база, нейтральные строения;
- экономика: получение ресурсов, списание ресурсов при найме и улучшениях;
- бой: отдельный экран, пошаговый расчет без тактической сетки и без позиционирования, порядок ходов по инициативе, выбор цели атаки, расчет урона, учет потерь, выдача награды;
- герой, армия, юниты, слоты армии, характеристики отрядов;
- база и найм: 1-3 постройки, пополнение найма раз в 7 ходов, накопление неиспользованного найма, минимум одно улучшение на постройку;
- сохранение и загрузка: состояние героя, армии, карты, ресурсов, пользовательских настроек;
- интерфейс: карта, бой, база/найм, окна информации, настройки.

Технологии:
- Unity LTS;
- C#;
- Unity UI (uGUI);
- TextMeshPro;
- JSON и Newtonsoft.Json;
- Git и GitHub;
- Rider или Visual Studio.

Целевая платформа: персональные компьютеры под управлением Windows 10/11.

Особые ограничения: игра автономная, без мультиплеера, облачных сохранений и внешних сервисов; боевой экран не содержит тактической сетки; несколько героев и активные AI-соперники не входят в объем проекта.
