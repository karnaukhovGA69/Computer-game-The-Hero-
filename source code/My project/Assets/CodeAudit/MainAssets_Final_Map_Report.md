# MainAssets Final Map Report

This file is updated when you run **The Hero → Map → Build Final Map From MainAssets** and **The Hero → Validation → Validate MainAssets Map** in the Unity Editor.

## Before first build

1. Open the project in Unity.
2. Run **The Hero → Map → Build Final Map From MainAssets**.
3. Run **The Hero → Validation → Validate MainAssets Map**.
4. Play → MainMenu → New Game and verify gameplay manually.

## Expected outcome

- Map size ~48×32 from MainAssets sliced sprites only (castle may come from elsewhere in the project).
- Castle at center (~24, 16), hero beside castle.
- Zones: center start, west forest, east stone, north dark boss, south road, river + bridge.
- Enemies: wolves, skeletons, skeleton mage, gargoyle/troll guard, Underworld King as Dark Lord.
- Resources: 4 gold, 3 wood, 3 stone, 2 mana, 2 chests, 1 artifact.
