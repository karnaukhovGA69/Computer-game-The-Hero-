THE HERO - DEMO REPORT

1. Реализовано:
   - Сюжетная кампания из 10 стадий.
   - Система диалогов и квестов.
   - Улучшенная карта 16x10 с логичными зонами и препятствиями.
   - Пошаговый бой с инициативой и автобоем.
   - Развитая база с карточками зданий, наймом и улучшениями.
   - Система сложности (Easy, Normal, Hard).
   - Мини-карта для обзора целей.
   - Главное меню с Помощью, Авторами и подтверждением новой игры.
   - Надежное сохранение и загрузка.

2. Как запустить в Unity:
   - Открыть Assets/Scenes/MainMenu.unity.
   - Нажать Play.

3. Как собрать .exe:
   - Меню The Hero -> Build -> Build Demo Windows EXE.

4. Где находится .exe:
   - Builds/TheHeroDemo/TheHero.exe.

5. Ограничения:
   - ИИ на карте отсутствует (враги статичны).
   - Графика использует процедурные спрайты.

6. Что улучшить:
   - Добавить тактическую сетку в бою.
   - Добавить больше типов юнитов и магию.
   - Улучшить визуальные эффекты.
