THE HERO - FINAL MVP VERSION

Инструкция:
1. Запустите Builds/TheHeroFinal/TheHero.exe.
2. В главном меню выберите 'Новая игра'.
3. Изучите обучение и помощь (кнопка Help).
4. Перемещайтесь по карте кликом мыши.
5. Захватывайте шахты и собирайте ресурсы для развития армии.
6. Развивайте базу (кнопка 'Замок' или объект на карте).
7. Победите Тёмного Лорда на северо-востоке для завершения игры.

Управление:
- Левая кнопка мыши: Движение, выбор отряда, атака.
- End Turn: Завершение хода.

Техническая информация:
- Сохранения находятся в Application.persistentDataPath.
- При ошибках загрузки игра попробует восстановиться из бэкапа.
