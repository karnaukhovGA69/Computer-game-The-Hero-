РУКОВОДСТВО ПОЛЬЗОВАТЕЛЯ - THE HERO (Release Candidate)

1. Начало игры
- Запустите 'TheHero.exe' или сцену 'MainMenu' в редакторе.
- Используйте 'New Game' для старта.

2. Мировая карта
- Клик по клетке - перемещение.
- Клик по объекту - взаимодействие.
- Ресурсы: золото, дерево, камень, мана.
- Шахта: захватите для пассивного дохода золота.
- База: кликните по иконке базы, чтобы зайти в замок.
- Враги: клик по врагу начнет бой.

3. Управление замком
- Нанимайте воинов в доступных зданиях.
- Улучшайте здания для открытия новых возможностей.
- 'Back To Map' - возврат на карту мира.

4. Сражения
- Пошаговый режим.
- Используйте 'Attack' для хода.
- Используйте 'Auto Battle' для быстрого завершения.

5. Системные клавиши
- Esc - Меню паузы (Сохранить, Загрузить, Выйти).
- End Turn - Закончить текущий день и восстановить очки хода.
